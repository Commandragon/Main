Build Your Own Game Site

Use this site at home: https://bit.ly/Codedragon

Step 1: Create Accounts

  • Replit - Write and test your code.
  • Github - Store your code
  • Render - Host your website.
  • Freedns - Get a free subdomain for your website.
  • ChatGPT (if you don't know how to code) - Get help.

Step 2: Get the Code

Copy the code for your website from these repositories:

  • On Replit: Code on Replit
  • On Github: Code on Github

Step 3: Add Games

Add games to your website:

  • Main website: 3kh0-lite Repo
  • Soundboard: Soundboard Repo

Step 4: Make Your Website Live

Use Render to make your website live. If you need help, join the Discord server: Discord.

Step 5: Get a Domain

Go to Freedns and get a free subdomain (choose a public domain if you want it to take less time) for your website. Make sure to choose a good name for your main website. Set the DNS to 216.24.57.1.

Step 6: Link Your Domain

Link your subdomain to your website on Render. Go to settings and find the option for custom domains.

Step 7: Final Touches

Add links to all your games on your main website. Edit the main page however you like, but remember to give credit to CodeDragon.
